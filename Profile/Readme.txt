Installation:

This project is base on Windows envorinment. Make sure you have Node and NPM installed on your computer before these steps. Next, at the command prompt, go to the project folder and type the following steps at the prompt:

1. bower install
2. npm install gulp --save-dev
3.npm install jshint gulp-jshint jshint-stylish gulp-imagemin gulp-concat gulp-uglify gulp-minify-css gulp-usemin gulp-cache gulp-changed gulp-rev gulp-rename gulp-notify  gulp-ng-annotate browser-sync del --save-dev

Then start the gulp server use following prompt:
1. gulp
2. gulp watch

Now you can see the page on the browser. (the address should be localhost:3000 ...)
